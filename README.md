# Kifo — Honor Every Life

A warm, beautifully designed online memorial platform. Create lasting memorial websites, share cherished memories, plan events, and bring family together from anywhere in the world.

## 🌿 Live Demo

Hosted on GitHub Pages: **[View Kifo →](https://YOUR-USERNAME.github.io/kifo)**

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — hero, features, events, pricing, testimonials |
| `kifo-memorial.html` | Individual memorial page — tributes, gallery, stories, events tabs |
| `kifo-wizard.html` | 4-step memorial creation wizard |
| `kifo-dashboard.html` | Admin dashboard — manage memorials, activity feed, reminders |
| `kifo-events.html` | Event planning — create services, gatherings, collect RSVPs |

## Design

- **Fonts:** Cormorant Garamond (display) + DM Sans (body)
- **Palette:** Terracotta, parchment, sage, dusk — warm & organic
- **Fully responsive** — mobile-first layouts on all pages
- **No frameworks** — pure HTML, CSS, vanilla JS

## Features

- 🕯 Online memorial pages with photo galleries, tributes, stories
- 🗓 Event planning (funeral services, celebrations of life, anniversaries)
- 🔒 Privacy controls — public or invite-only
- 🎵 Background music playlist
- 📱 QR code support for services
- 🔔 Anniversary & birthday reminders
- 🌍 Unlimited family collaborators

## Getting Started

```bash
git clone https://github.com/YOUR-USERNAME/kifo.git
cd kifo
# Open index.html in your browser, or serve locally:
python3 -m http.server 8080
# Then visit http://localhost:8080
```
